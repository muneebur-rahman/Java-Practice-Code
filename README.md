# Java-Practice-Code

A collection of structured Java programs covering core Java, OOPs, arrays, and strings for hands-on practice and skill development.

This repository contains structured Java practice programs created to strengthen Core Java fundamentals.
Programs are organized into different folders based on concepts for better understanding and practice.

## Folder Structure

- _basics/_

  - HelloWorld
  - Data-Types
  - Conditional statement
  - Loops
  - Scanner
  - Project

- _oops/_

  - InheritanceDemo.java
  - PolymorphismDemo.java
  - InterfaceDemo.java

- _arrays/_

  - ArraySum.java
  - TwoDArray.java

- _strings/_
  - ReverseString.java
  - Palindrome.java

## Topics Covered

- Core Java Basics
- Control Statements and Loops
- Object-Oriented Programming (OOPs)
- Arrays and Strings

## Technologies Used

- Java (Core Java)
- JDK
- VS Code / Any Java-supported IDE

## Purpose

This repository is created for learning and practice purposes.
It helps in building strong Java fundamentals through hands-on coding.
